
# US Retail Sales and Holiday Impact Analysis

This project analyzes how public holidays affect retail sales in the US.

## Overview
We explore a dataset of daily retail orders and public holidays from 2014 to 2017. The goal is to understand:

- How daily orders change over time.
- Whether holidays affect the number of orders.
- Which holidays have the biggest impact on sales.

## Features
- Load and clean integrated sales and holiday data.
- Calculate daily orders and compare holidays vs non-holidays.
- Identify top holidays by number of orders.
- Visualize trends using line plots.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Insights
- Average daily orders on holidays vs non-holidays.
- Holidays ranked by total sales impact.
- Daily order trends over multiple years.

## How to Run
1. Clone the repository.
2. Place the `orders_integrated_holidays.csv` file in the `DATA/processed/` folder.
3. Run `Retail_Holiday_Analysis.ipynb` in Jupyter Notebook.
